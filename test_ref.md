### SQL

~~~
SELECT column_names 
FROM table_name
~~~

~~~
SELECT' * 
FROM table_name
~~~

~~~
SELECT DINSTINCT column_name 
FROM table_name
~~~ 
selects only the unique values from a table

~~~
SELECT column_name(s)
FROM table_name
